# hockeyteam
# Hockey Team Management API

RESTful API для управления хоккейной командой. Проект позволяет управлять игроками, командами, позициями, достижениями и статистикой.

## 🏒 О проекте

API разработано для автоматизации управления хоккейной командой. Система позволяет:
- Управлять профилями игроков
- Распределять игроков по командам и позициям
- Отслеживать статистику игроков (голы, передачи, игры)
- Назначать достижения игрокам
- Вести историю сезонов

## 🛠 Технологический стек

- **Java 21**
- **Spring Boot 4.0.3**
- **Spring Data JPA** - работа с базой данных
- **PostgreSQL** - база данных
- **MapStruct** - маппинг DTO и Entity
- **Lombok** - генерация кода
- **Maven** - сборка проекта

## 📋 Модели данных

### Player (Игрок)
- Личные данные (имя, фамилия, возраст, номер)
- Статистика (голы, передачи)
- Связи с командой, позицией, достижениями

### Team (Команда)
- Название
- Город
- Список игроков

### Position (Позиция)
- Название позиции (нападающий, защитник, вратарь)

### Achievement (Достижение)
- Название
- Описание
- Игроки, получившие достижение

### Statistic (Статистика)
- Статистика по сезонам
- Голы, передачи, игры
